Agent 七种执行模式

由 Agent.execution_mode 决定 · AgentExecutor 统一调度 · 三引擎复用 · LangFuse 自动追踪

Agent 配置
execution_mode
→ AgentExecutor.execute() → 7 种执行器 → LangFuse Span
⚡ single_pass 3-10s
单次 LLM 调用 — 简单问答、单函数实现
[⚡ single_pass] 前端工程师-Agent (frontend_dev) ├── input: 用户 prompt ├── output: LLM 回复内容 └── chat:deepseek-v4-pro (tokens + latency)
🔗 chain_of_thought 5-15s
思维链推理 — 复杂问题分步思考
[🔗 chain_of_thought] 产品经理-Agent (pm) ├── input: 用户 prompt（含 CoT 引导） ├── output: 推理过程 + 最终答案 └── chat:deepseek-v4-pro (tokens + latency)
📋 plan_execute 20-60s
三阶段规划执行 — Plan → Review → Supplement
[📋 plan_execute] 产品经理-Agent (pm) ├── input: 任务描述 ├── output: 完整规划 + 补充结果 ├── [Phase 1: Plan] → chat:deepseek (plan) ├── [Phase 2: Review] → chat:deepseek (review) └── [Phase 3: Supplement] → chat:deepseek (supplement)
🔄 ReAct 20-120s
Think → Act → Observe 循环 — 工具调用
[🔄 react] 后端工程师-Agent (backend_dev) ├── input: 任务 + 工具声明 ├── output: 最终答案 ├── [Iteration 1] │ ├── chat:deepseek (Think) │ └── tool:write_file (Act) ├── [Iteration 2] │ └── chat:deepseek (Think + final) └── ... (max 5 iterations)
📦 ReWOO 15-60s
规划 + 并行工具执行 + 合并结果
[📦 rewoo] 部署运维-Agent (devops) ├── input: 任务描述 ├── output: 合并结果 ├── [Phase 1: Plan] → chat:deepseek (plan) ├── [Phase 2: Execute] → tool:write × N (parallel) └── [Phase 3: Merge] → chat:deepseek (merge)
🪞 Reflexion 30-180s
自我评分 + 根据反馈迭代改进
[🪞 reflexion] 测试员-Agent (tester) ├── input: 任务 + 测试标准 ├── output: 最终代码（经过 self-critic） ├── [Round 1: Execute] → chat:deepseek (initial) ├── [Critique Round 1] → chat:deepseek (score+issues) ├── [Redo Round 2] → chat:deepseek (improved) └── ... (until score ≥ threshold or max rounds)
🗳️ self_consistency 30-120s
3 次独立采样 → 综合最优方案
[🗳️ self_consistency] 架构师-Agent (architect) ├── input: 任务（含采样引导） ├── output: 综合后的最终方案 ├── [Phase 1: Sampling] │ ├── chat:deepseek (sample 1/3) │ ├── chat:deepseek (sample 2/3) │ └── chat:deepseek (sample 3/3) └── [Phase 2: Merge] → chat:deepseek (synthesis)
