📦 资源依赖链

Model → Persona + Tool → Agent → Skill → Team → SOP
Agent = Model + Persona + Tools  |  Skill 独立文件目录  |  SOP 节点绑定 Agent + Skill

🔄 核心执行流程

1. 注册资源 → 2. 创建团队 → 3. 定义SOP → 4. 分配Agent → 5. 节点执行 → 6. 验证+反馈 → 7. HITL确认 → 8. 任务完成
💡 每个节点执行时：加载Skill → 注入Memory+RAG → Agent调用 → 验证 → 条件路由
