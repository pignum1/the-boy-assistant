# 安全审查 Checklist

## OWASP Top 10 (2021)

### A01: 访问控制失效
- [ ] 所有 API 都有鉴权中间件
- [ ] 资源级别权限校验（不只是功能级别）
- [ ] CORS 配置正确（不允许 `*`）
- [ ] 敏感操作有二次确认

### A02: 加密失败
- [ ] 传输层使用 HTTPS
- [ ] 密码使用 bcrypt/argon2 哈希
- [ ] 敏感数据不记录到日志
- [ ] API Key 不硬编码在代码中

### A03: 注入
- [ ] 使用参数化查询（非字符串拼接 SQL）
- [ ] 用户输入经过校验和转义
- [ ] 命令执行不拼接用户输入

### A04: 不安全设计
- [ ] 有限流保护
- [ ] 批量操作有上限
- [ ] 不暴露内部 ID 结构

### A05: 安全配置错误
- [ ] 关闭 DEBUG 模式
- [ ] 移除默认账户和密码
- [ ] 错误信息不暴露堆栈

### A06: 脆弱组件
- [ ] 依赖版本无已知漏洞
- [ ] 定期更新依赖

### A07: 鉴权失败
- [ ] JWT 有过期时间
- [ ] Refresh Token 有轮转机制
- [ ] 登出使 Token 失效

### A08: 数据完整性
- [ ] 反序列化有限制
- [ ] CI/CD pipeline 有签名验证

### A09: 日志监控
- [ ] 记录关键操作审计日志
- [ ] 异常有告警通知

### A10: SSRF
- [ ] 用户提供的 URL 有白名单校验
- [ ] 不允许请求内网地址
