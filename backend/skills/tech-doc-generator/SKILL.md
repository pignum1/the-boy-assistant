# 技术文档生成器

生成专业的技术文档，包括 ADR、API 设计、技术方案等。

## 支持的文档类型

1. **ADR (Architecture Decision Record)** — 架构决策记录，包含：标题、状态、上下文、决策、后果
2. **技术方案文档** — 背景、目标、方案对比、详细设计、风险评估、排期
3. **API 设计文档** — 接口清单、请求/响应格式、错误码、鉴权方式
4. **数据库设计文档** — ER 说明、表结构字典、索引策略、迁移计划
5. **README / 项目文档** — 项目介绍、快速开始、架构概览、贡献指南

## 输出格式

- **纯 Markdown** — 可直接放入项目仓库
- **带 Mermaid 图表的 Markdown** — 架构图、流程图、时序图内嵌

## 工作流程

1. 确定文档类型和受众
2. 收集必要信息（代码、架构、接口定义）
3. 按模板结构组织内容
4. 生成文档，确保可读性和完整性

## 质量标准

- ADR 必须包含：决策原因、备选方案及对比、预计影响
- 技术方案必须包含：风险矩阵和回滚预案
- API 文档必须包含：每个接口的请求/响应示例
- 所有文档使用中文撰写，技术术语保留英文
