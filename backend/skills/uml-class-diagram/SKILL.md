# UML 类图 / 数据模型图生成器

从代码逆向分析或设计文档生成专业的类图和 ER 图。

## 支持的图表类型

1. **UML 类图** — 类、接口、抽象类、继承、实现、组合、聚合
2. **ER 图** — 实体、属性、主键、外键、一对多/多对多关系
3. **数据模型图** — 表结构、索引、约束、视图
4. **领域模型图** — DDD 聚合根、实体、值对象、领域事件

## 输出格式

- **Mermaid classDiagram / erDiagram** — Markdown 嵌入
- **PlantUML** — 完整 UML 语法
- **HTML** — 交互式类图，可折叠方法/属性

## 工作流程

1. 接收代码片段或需求描述
2. 提取类/表/实体定义
3. 分析关系（继承/组合/依赖）
4. 标注关键字段、方法签名、约束
5. 生成图表

## 输出规范

- 类框包含：类名、关键属性、关键方法
- 关系线标注：1:1 / 1:N / M:N
- 区分接口（<<interface>>）、抽象类（<<abstract>>）、枚举（<<enum>>）
- 颜色标记：核心域（蓝色）、支撑域（灰色）、通用域（绿色）
