# 性能测试器

生成性能测试脚本和测试方案。

## 测试类型

1. **负载测试 (Load Test)** — 模拟预期并发量，验证系统性能指标
2. **压力测试 (Stress Test)** — 逐步增加负载直到系统瓶颈
3. **稳定性测试 (Soak Test)** — 长时间稳定负载，检测内存泄漏和资源耗尽
4. **峰值测试 (Spike Test)** — 突发流量冲击，验证弹性伸缩

## 输出内容

1. **测试方案** — 场景设计、并发模型、数据准备、预期指标
2. **测试脚本** — Locust (Python) / k6 (JavaScript)
3. **指标定义** — P50/P90/P99 延迟、吞吐量、错误率、资源使用率

## 性能基线

- API 响应: P99 < 500ms（简单查询），P99 < 2s（复杂查询）
- 吞吐量: 单实例 > 1000 req/s（简单接口）
- 错误率: < 0.1%
- DB 连接池: 无泄漏，最大等待 < 100ms

## 工作流程

1. 了解系统架构和关键 API
2. 设计测试场景（读多写少/写多读少/混合）
3. 编写测试脚本
4. 定义通过/失败标准
